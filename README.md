# Employee Payroll Management System

## Summary
A Core Java console-based Employee Payroll Management System that manages employee records and payroll operations using a layered architecture.

---

# Overview

Employee Payroll Management System is a backend-focused Java application developed to manage employee information and payroll records through a console interface.

The project follows a clean layered architecture consisting of:
- Model layer
- DAO layer
- Service layer
- Database utility layer

The system demonstrates object-oriented programming principles and separation of concerns while implementing employee management and payroll-related operations.

---

# Problem Statement

Managing employee and payroll information manually can lead to:
- Data inconsistency
- Difficult record maintenance
- Calculation errors
- Poor code organization

This project solves these issues by providing:
- Centralized employee management
- Structured application layers
- Organized payroll operations
- Maintainable and reusable code architecture

through a Java-based console application.

# Methods

## Model Layer
Contains entity classes representing employee and payroll data.

## DAO Layer
Handles data access and performs CRUD operations on employee records.

## Service Layer
Implements business logic and coordinates communication between layers.

## Database Utility Layer
Provides centralized data handling and storage management.

## Main Class
Acts as the entry point of the application and provides the console interface.

# How to Run Project

## Clone Repository

```bash
git clone https://github.com/Sham1718/EmployeePayroll.git
```

---

## Navigate to Project

```bash
cd EmployeePayroll
```

---

## Open Project

Open the project in:
- IntelliJ IDEA
- Eclipse
- VS Code

---

## Compile and Run

Run the `Main.java` file:

```bash
javac Main.java
java Main
```

Or simply run `Main.java` directly from your IDE.
